   ```bash

   ```

   ```bash
   pip install -r requirements.txt
   ```

   The project needs data to run. Generate it using the included script:
   ```bash
   python generate_data.py
   ```

5. **Run the Dashboard**:
   ```bash
   streamlit run app.py
   ```
